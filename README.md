# plus-smart-home-tech

## Порядок запуска

1. Запустить докер контейнер
- Выполнить команду в папке с проектом: "**docker-compose up -d**"

2. Запустить модуль **discovery-server**
3. Запустить модуль **config-server**
4. Запустить модуль **gateway**
5. Запустить модуль **warehouse**
6. Запустить модуль **shopping-store**
7. Запустить модуль **shopping-cart**
8. Запустить модуль **payment**
9. Запустить модуль **order**
10. Запустить модуль **delivery**
